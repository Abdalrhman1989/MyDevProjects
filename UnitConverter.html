Length Converter

1. Variabler

Variabler bruges til at gemme data. I denne enhedsomregner bruger vi variablen unitFactors til at gemme konverteringsfaktorerne.

// Variabel til konverteringsfaktorer
const unitFactors = {
meters: 1,
centimeters: 100,
millimeters: 1000,
kilometers: 0.001,
inches: 39.3701,
feet: 3.28084,
yards: 1.09361,
miles: 0.000621371
};

2. Datatyper

Forskellige datatyper bruges i dette projekt. Her bruger vi primært objekter og tal:

  • Object: Bruges til at gemme konverteringsfaktorerne.
  • Number: Bruges til at gemme inputværdien og konverteringsresultatet.
// Eksempel på brug af objekter og tal
const inputValue = document.getElementById('inputValue').value; // Nummer
const result = (inputValue * unitFactors[inputUnit] / unitFactors[outputUnit]).toFixed(4); // Nummer

4. Betingelser

Betingelser gør det muligt for os at udføre visse kodeblokke baseret på specifikke betingelser. I denne enhedsomregner bruger vi betingelser til at kontrollere brugerinput og beregne resultatet.

// Funktion til at konvertere enhed
function convert() {
const inputValue = document.getElementById('inputValue').value;
const inputUnit = document.getElementById('inputUnit').value;
const outputUnit = document.getElementById('outputUnit').value;
if (inputValue && inputUnit && outputUnit) {
const result = (inputValue * unitFactors[inputUnit] / unitFactors[outputUnit]).toFixed(4);
document.getElementById('result').innerText = `${inputValue} ${inputUnit} is ${result} ${outputUnit}`;
}
}
