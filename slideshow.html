1. Variabler

Variabler bruges til at gemme data. I dette slideshow bruger vi variablen slideIndex til at holde styr på den aktuelle slide. En variabel er som en beholder, der holder information, som kan ændre sig over tid.

// Deklaration af en variabel
let slideIndex = 1;

// Brug af variablen til at vise den første slide
showSlides(slideIndex);

2. Datatyper

Forskellige datatyper bruges i dette projekt. Her bruger vi primært tal og objekter:

  • Number: Bruges til at gemme indekset for den aktuelle slide.
  • Object: HTML-elementerne, der repræsenterer slides og prikker.
// Brug af number til at gemme slide-indekset
let slideIndex = 1;

// Brug af object til at gemme HTML-elementer
let slides = document.getElementsByClassName("mySlides");
let dots = document.getElementsByClassName("dot");

4. Betingelser

Betingelser gør det muligt for os at udføre visse kodeblokke baseret på specifikke betingelser. I dette slideshow bruger vi betingelser til at sikre, at vi skifter korrekt mellem slides.

function showSlides(n) {
let i;
let slides = document.getElementsByClassName("mySlides");
let dots = document.getElementsByClassName("dot");

// Hvis n er større end antallet af slides, start fra første slide
if (n > slides.length) {slideIndex = 1}

// Hvis n er mindre end 1, start fra sidste slide
if (n < 1) {slideIndex = slides.length}

// Skjul alle slides
for (i = 0; i < slides.length; i++) {
slides[i].style.display = "none";
}

// Fjern "active" klassen fra alle prikker
for (i = 0; i < dots.length; i++) {
dots[i].className = dots[i].className.replace(" active", "");
}

// Vis den aktuelle slide og tilføj "active" klassen til den tilsvarende prik
slides[slideIndex-1].style.display = "block";
dots[slideIndex-1].className += " active";
}
